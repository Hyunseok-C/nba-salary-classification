### 🎯 연구 주제

---

### 🔍 분석 방법
- 연봉 구간화 - 샐러리캡 비율 기준 (`Low`: 13% 이상, `Mid`: 13% 미만, `High`: 2% 미만)
- 학습 데이터 80%, 검증 데이터 20%로 5-Fold 교차검증 수행
- 변수 선택: Lasso 회귀
  
| 모델 | 세부 설정 및 방법 |
|:------:|:----------------|
| **KNN** | 커널: `gaussian` 거리: `Manhattan` (p=1) |
| **Naive Bayes** | 변수의 분포 왜도(`skewness`)에 따라 Quantile(사분위) 또는 Equal Width(등간격) 범주화 적용 |
| **결정트리** | 완전성장트리 → 가지치기로 과적합 방지 |
| **랜덤포레스트** | 트리 수: `B=500`, Out-of-Bag Error(OOB)로 모델 성능 평가 |


---

### 🧠 분석 결과
- 모델 별 성능
  - 전반적 안정성과 해석력: 랜덤포레스트 모델
  - 최고 정확도: KNN 모델
  - **랜덤포레스트 모델**이 예측 정확도와 안정성 측면에서 가장 우수  
- 핵심지표
  - 경력(Year.of.Service), 경기당 평균 득점 수(PPG), 대체선수 대비 기여도(VORP), 선발 출전한 경기수(GS)
- 인사이트
  - 과대 예측: Darius Garland 선수 (`Mid` -> `High`)
    - 시사점: 루키 계약으로 인해 고성과-저연봉의 사례 (가성비 인재)  
  - 과소 예측: Al Horford 선수 (`High` -> `Mid`)
    - 시사점: 현재 시즌 성과만으로 베테랑 가치판단이 어려움
